# Security Policy / Política de Segurança

_(Português abaixo / Portuguese below)_

---

## Supported Versions

Currently, only the latest active development branch (`main`) is supported with security updates. We recommend keeping your clone/deployment up to date.

| Version | Supported          |
| ------- | ------------------ |
| v1.x    | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability

Please do not open a public issue for security vulnerabilities. Instead, responsibly disclose it by sending an email to **<samuelaoliveiraramos@gmail.com>**.

We will review the report and respond within 48 hours. If the vulnerability is confirmed, we will work on a patch and release it as soon as possible, giving you credit in the release notes.

## Relatando uma Vulnerabilidade

Por favor, não abra uma issue pública para relatar vulnerabilidades de segurança. Em vez disso, faça a divulgação responsável enviando um e-mail para **<samuelaoliveiraramos@gmail.com>**.

Nós revisaremos o relatório e responderemos em até 48 horas. Se a vulnerabilidade for confirmada, trabalharemos em um patch e o lançaremos o mais rápido possível, dando a você os créditos nas notas de lançamento.
